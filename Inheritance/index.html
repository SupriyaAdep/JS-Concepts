Inheritance

Objects and Functions

  • Every JS object has a prototype property which makes inheritance possible in JS
  • The prototype property of an object is where we put method and properties that we want other objects to inherit
  • The constuctor's prototype property is not the prototype property of the constructor itself, it's the prototype of all instances that are created through it
  • When a certain method or property is called, the search starts in the object itslef and if cannot be found the search moves on to object prototype prototype. This continues until the method is found in the prototype chain
